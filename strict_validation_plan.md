# 严格验证计划：论文关键数据复现

## 验证目标

严格验证论文中的5个关键数值，确保计算精度和可重复性：

1. **αK值**: 20.9% ≤ αK ≤ 21.4% (氟原子探针方法)
2. **介电常数**: ε∞ = 3.80 ± 0.1
3. **vdW C60带隙**: Eg ≈ 2.0 eV
4. **IPR比值**: vdW=34, qHP=30 (±5%)
5. **热重整化**: ΔEg(T) = 0.10-0.16 eV

## 验证策略

### 阶段1：基础验证 (2-3天)
- 解决技术问题（rVV10兼容性）
- 建立基准计算（单分子→小超胞→完整超胞）
- 参数收敛性测试

### 阶段2：核心验证 (5-7天)  
- αK值精确确定
- 介电常数计算
- 带隙计算与实验对比

### 阶段3：高级验证 (3-5天)
- IPR局域化分析
- 热重整化MD模拟
- 误差分析和不确定度评估

## 数值精度要求

| 参数 | 目标精度 | 验证方法 |
|------|----------|----------|
| αK | ±0.1% | 多点线性拟合 |
| ε∞ | ±0.05 | 电场响应计算 |
| Eg | ±0.1 eV | DOS分析+实验对比 |
| IPR | ±10% | 统计分析多构型 |
| ΔEg(T) | ±0.02 eV | 长时间MD平均 |
